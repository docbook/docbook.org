The DocBook Namespace

The page describes the DocBook namespace, http://docbook.org/ns/docbook. This namespace identifies a family of documents that may be valid according one or more schemas.

RELAX NG Schemas

There is no normative schema for any language in this namespace at this time. The DocBook Technical Committee expects to publish the following schemas soon:

  • DocBook V5.0, a normative RELAX NG Grammar for DocBook V5.0.

W3C XML Schemas

There is no XML Schema for any language in this namespace at this time. The DocBook Technical Committee expects to publish the following schemas soon:

  • DocBook V5.0, an informative W3C XML Schema for DocBook V5.0.

XML DTDs

There is no DTD for any language in this namespace at this time. The DocBook Technical Committee expects to publish the following DTDs soon:

  • DocBook V5.0, an informative XML DTD for DocBook V5.0.
